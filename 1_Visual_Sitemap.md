## Squarespace Navigation Setup Instructions

1. **Create Folder Structure:**
   - Services (Folder)
   - Industries (Folder)
   - Products & Partners (Folder)
   - About (Folder)

2. **Set Dropdown Behavior:**
   - Enable "Simple" dropdown menus
   - First item in each folder = overview page
   - Hover reveals other pages

3. **Mobile Menu:**
   - Enable accordion-style dropdowns
   - Collapsible sections for each main nav item

4. **Footer:**
   - Create footer navigation with 4 columns
   - Manually select most important links
   - Add social media icons
   - Include contact info

## URL Structure Best Practices

### Keep URLs Clean & SEO-Friendly:
✅ Good: `/industries/healthcare`
✅ Good: `/products/partners/om-seating`
✅ Good: `/services/ergonomic-assessments`

❌ Avoid: `/page-1`
❌ Avoid: `/industries/healthcare-solutions-page`
❌ Avoid: `/om-seating-commercial-ergonomic-furniture`

### Squarespace Auto-Generation:
- Set custom URL slugs for each page
- Use hyphens, not underscores
- Keep URLs short but descriptive
- Match page titles when possible
