URL Structure Best Practices

Keep URLs Clean & SEO-Friendly:

✅ Good: /industries/healthcare
✅ Good: /products/partners/om-seating
✅ Good: /services/ergonomic-assessments

❌ Avoid: /page-1
❌ Avoid: /industries/healthcare-solutions-page
❌ Avoid: /om-seating-commercial-ergonomic-furniture

Squarespace Auto-Generation:

  • Set custom URL slugs for each page
  • Use hyphens, not underscores
  • Keep URLs short but descriptive
  • Match page titles when possible
