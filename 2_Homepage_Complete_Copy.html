Mobile Optimization Notes

Hero Section:

  • Stack text over image
  • Reduce headline font size
  • Stack buttons vertically
  • Ensure buttons are thumb-friendly (min 44px height)

What We Do Cards:

  • Stack 3 cards vertically
  • Full width on mobile
  • Maintain readable padding

Industry Grid:

  • 1 column on mobile
  • Cards remain clickable
  • Maintain image quality

All Sections:

  • Increase padding for readability
  • Ensure minimum 16px font size
  • Test tap targets (min 44x44px)
  • Optimize image loading (lazy load)

Color & Typography Specifications

Primary Colors:

  • Brand Primary: #2C5F7C (deep blue) or your chosen color
  • Brand Secondary: #E67E50 (warm orange for CTAs)
  • Neutral Dark: #333333 (body text)
  • Neutral Light: #F5F5F5 (backgrounds)
  • White: #FFFFFF

Typography:

  • Headlines (H1): 48px desktop / 32px mobile, Bold
  • Section Headers (H2): 36px desktop / 28px mobile, Bold
  • Subheadings (H3): 24px desktop / 20px mobile, Semi-bold
  • Body Text: 18px desktop / 16px mobile, Regular
  • Button Text: 16px, Semi-bold, Uppercase

Font Families:

  • Headers: Montserrat, Poppins, or similar (bold, modern)
  • Body: Inter, Open Sans, or similar (clean, readable)

Squarespace Implementation Tips

  1. Use Spacer Blocks between sections for consistent vertical rhythm
  2. Enable Parallax sparingly on hero image only
  3. Set Anchor Links for smooth scrolling from nav
  4. Use Grid Blocks for industry cards and partner logos
  5. Enable Mobile Styles and preview on all devices
  6. Add Alt Text to all images for SEO and accessibility
  7. Compress Images before uploading (use TinyPNG)
  8. Test CTAs - ensure all buttons link correctly
